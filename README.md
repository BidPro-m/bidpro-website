# BidPro Corporate Website — GitHub Pages

This package contains the production-ready static website for BidPro and the Procurement Intelligence Research Programme 2026.

## Pages
- `index.html` — corporate homepage
- `products.html` — BidPro platform vision
- `research.html` — full research programme landing page
- `publications.html` — executive publication downloads
- `about.html` — BidPro overview
- `contact.html` — programme contact details

## Publish with GitHub Pages
1. Create a new public repository named `bidpro-website` under the BidProSA GitHub organisation.
2. Upload all files and folders from this package to the repository root.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select branch `main` and folder `/ (root)`.
6. Click **Save**.
7. GitHub will provide a public URL such as `https://bidprosa.github.io/bidpro-website/`.
